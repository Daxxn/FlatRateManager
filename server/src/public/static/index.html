Flat Rate Tracker Server

Simplified version of the `Repair Order Tracker` ( Client & Server ) . The full project is still on the books but it needs some major changes and I need to get this up and running. I let scope creep get its teeth into it. This is the lean mean version.

Some of the main design goals:

  • Fast and easy data entry
  • Simple data structure that can be easily extended
  • Metrics for graphing and displaying times for each job or person
